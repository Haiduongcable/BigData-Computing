# BigData-Computing

## Dependencies 
* [Python 3.7](https://www.python.org/download/releases/3.7/)

```
$ pip install -r requirements.txt
```

## Xây dựng spark cluster
```
$ cd Spark-Cluster
$ docker-compose up
```

## Thực hiện thống kê dữ liệu

```
$ cd Data_Process
$ python char_review_point.py
```
